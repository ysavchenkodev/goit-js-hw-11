Кожну секунду треба:

брати поточний час: Date.now() знаходити різницю: const diff =

Ця функція повинна повернути:

10. Створи функцію форматування

Напиши функцію, наприклад:

function addLeadingZero(value) { return String(value).padStart(2, "0"); }
